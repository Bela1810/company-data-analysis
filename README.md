# Employee Attrition & Performance Analysis (EDA)

## 📌 Project Overview
This project performs a comprehensive **Exploratory Data Analysis (EDA)** on a Human Resources dataset to understand the factors that lead to employee attrition. The goal is to uncover patterns related to job satisfaction, compensation, and work-life balance to provide actionable business insights.

## 📊 Dataset Features
The dataset contains **1,499 records** and **35 columns**, including:
*   **Target Variable:** `Attrition` (Whether an employee left the company).
*   **Demographics:** Age, Gender, Marital Status, Education.
*   **Professional Profile:** JobRole, JobLevel, TotalWorkingYears, YearsAtCompany.
*   **Compensation & Performance:** MonthlyIncome, PercentSalaryHike, PerformanceRating.
*   **Satisfaction Levels:** EnvironmentSatisfaction, JobInvolvement, RelationshipSatisfaction, WorkLifeBalance.

## 🛠️ Tech Stack
*   **Language:** Python
*   **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
*   **Environment:** Jupyter Notebook

## 🔍 Key Objectives
1.  **Data Cleaning:** Handle missing values in columns like `Attrition`, `MonthlyIncome`, and `TotalWorkingYears`.
2.  **Univariate Analysis:** Distribution of age, salary, and tenure.
3.  **Bivariate Analysis:** Relationship between `OverTime` and `Attrition`, and how `MonthlyIncome` varies across `JobRole`.
4.  **Correlation Study:** Identifying which factors have the strongest impact on staff turnover.
